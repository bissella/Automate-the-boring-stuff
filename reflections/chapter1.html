Reflections from Journey Learning Python through Automate the Boring Stuff

What do I want to learn?

Python always reduces an expression to a single value.
Putting junk into the console is fine. No problem with getting an error message.


The lowest common denominator in python is:

- Values
- Expressions
- Operators

All expressions simplify to a single value with operators. There are only a few types of data that a final value can be.
Unlike HTML and other programs, hypens and dashes are out. Underscores are the way to go!

What was useful?

input() prompts for input from the user in the console.

spam = int(input())

Flow Control

What do I want to learn?

All flow control statements start with a condition which is an expression that always results in a boolean term (True or False). Code blocks follow control statements and use indentation to signal where they start and end.

What was useful?

Diamonds represent branching points of the flow control. Usually a question with different answers flowing out.
Python has 'not' reserved as a comparison term. It's the same as the negative of a logic statement. I.e. not False == True
If else statments are the same as they are in R.
Elif
Elif comparisons put a following if to the else statment. Important note is that in a series of elif statments only one will be true or they will all be false. This means that after the first elf statement is true, the rest are ignored. Best practice is to end an elif series with an else statement.

While Loops

  • This will be useful for optimization algorithms!
  • While loops need a negative to break the loop.
